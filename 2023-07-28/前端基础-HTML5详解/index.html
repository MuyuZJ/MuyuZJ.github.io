初识HTML

什么是HTML

HTML 即Hyper Text Markup Language 超文本标记语言 超文本 包括了文字, 图片, 音视频, 动画等多种元素

HTML描述了网页的结构和内容 并通过使用标签来定义文本 图像 链接和其他元素的显示方式 HTML使用标记（尖括号包围的名称）来标记文本 这些标记告诉浏览器如何显示文本或者执行其他操作

HTML发展历史

HTML（超文本标记语言）的发展可以追溯到20世纪90年代

  • 1993年: HTML的第一版问世 标记了Web技术的起点
  • 1995年: HTML2.0发布 引入了一些新的标签和特性
  • 1997年: HTML3.2出现 给网页设计师带来了更多的布局和样式控制选项
  • 1999年: HTML4.01成为W3C（万维网联盟）推荐的标准 增加了更多的语义化标签和表单控件
  • 2000年: XHTML1.0发布 是HTML与XML的结合 强调正确的文档结构和规范的编码
  • 2008年: HTML5成为W3C推荐的标准 引入了许多新的特性和API 如视频 音频 Canvas绘图等
  • 2014年: HTML5.1发布 对HTML5进行了一些修订和扩展
  • 2016年: HTML5.2发布 进一步完善了HTML5的规范 HTML的发展历史中 每一个版本都带来了新的标签 特性和能力 使得Web应用程序能够更加丰富和交互 HTML5成为了现代Web开发的主要标准 为开发人员提供了更多的工具和功能 使其能够更好地构建复杂的Web应用程序

什么是HTML5

HTML5是HTML的第五个主要版本 是万维网联盟（W3C）推荐的标准 它引入了许多新的特性和API 为开发者提供了更多的工具和功能 使其能够构建更强大 更丰富的Web应用程序和网页

以下是HTML5的一些主要优势:

  1. 语义化标签: HTML5引入了一些新的语义化标签 如<header> <nav> <section> <article>等 这些标签使得开发者能够更清晰地定义网页的内容结构 提高了网页的可读性和可访问性
  2. 多媒体支持: HTML5提供了内置的多媒体支持 包括新增的<video>和<audio>标签 以及相关的API 这使得在网页上嵌入和管理视频 音频内容变得更加简单和灵活
  3. Canvas绘图: HTML5引入了<canvas>元素 它提供了一个用于绘制图形 动画和图表的空白画布 开发者可以使用JavaScript在画布上进行绘制 从而实现交互式的图形效果
  4. 本地存储: HTML5为客户端存储提供了多种选择 如localStorage和sessionStorage 这些功能允许开发者在用户的浏览器端存储和获取数据 提高了Web应用程序的性能和用户体验
  5. 支持离线应用: HTML5的应用缓存（Application Cache）功能使得开发者能够将网页在用户离线时仍然可访问 通过定义缓存清单文件 浏览器可以在用户离线时加载并显示已缓存的页面 提供更好的离线体验
  6. WebSocket: HTML5引入了WebSocket API 使开发者能够在浏览器和服务器之间建立持久的 双向的通信连接 这种实时通信机制为开发实时聊天 多人协作和游戏等应用提供了便利
  7. Web Workers: HTML5中的Web Workers API允许在后台运行脚本 不会影响页面的响应性能 开发者可以将一些耗时的任务放在Web Workers中运行 提高了Web应用的性能和用户体验

使用HTML5能让开发者能够更灵活 更高效地构建现代化的Web应用程序 并提供更丰富的用户体验

W3C标准

W3C 即 World Web Consortium, 万维网联盟 是Web领域最具有影响力和最权威的中立型技术标准机构

参考链接 w3c官方链接 w3c官方中文链接

W3C标准包括: 结构化标准语言(HTML XML) 表现标准语言(CSS) 行为标准(DOM ECMAScript)

HTML基本结构

以下是一个HTML文件的基本格式

  • 在首行说明文档类型 告诉浏览器使用哪个HTML版本来解析文档 被称为DTD文档类型声明
  • <head>标签用于定义文档的元数据和引入外部资源 如CSS样式表和JavaScript脚本
    • meta标签 用于声明一些内容 SEO(搜索引擎优化) 编写描述性信息等
    • title标签 定义标题
  • <body>标签用于定义文档的可见内容 如文本 图像和链接等

在HTML的body标签中 我们使用各种标签来标记不同的元素 如<h1>标签用于定义一个一级标题 <p>标签用于定义一个段落 <img>标签用于插入图像等

HTML还支持超链接（使用<a>标签）和表单（使用<form>标签和其他表单元素） 以实现用户与网页的交互

网页基本标签

介绍部分基本的网页标签 参考W3Cschool

单标签和双标签

  • 部分标签需要成对使用 称为双标签 例如<div></div>
  • 部分标签只需要单个标签即可表示对应的意思 称为单标签 例如 <br/>

块级元素与行内元素

  • 块级元素会独占一行 其他内容会换行显示 例如p标签
  • 行内元素不会独占一行 多个行内元素可以占用同一行 例如a标签

标题标签

在HTML中 h1h6为标题标签 作为网页标题 且只有h1h6有效 会自动换行 不同的标题标签样式不同语义也不同

段落标签

段落标签用于显示一个段落 会自动换行

换行标签

br标签是换行标签 可以实现内容的换行展示

水平线标签

用于展示一根水平线

字体样式标签

字体样式标签用于传统HTML网页控制字体的样式 后续建议使用css实现

特殊符号

在HTML文档中 部分符号无法直接输入 需要使用转义字符或者特殊符号表示

其他标签也可以参考W3Cschool的教程

图像

img标签 即为图像标签 可以在标签中添加对应的属性和值

img标签是块级元素 独占一行

  • src 必须参数 指定图片地址 可以是网络上的图片地址或者本机的图片地址
  • alt 定义 图片加载失败显示的文本
  • title 鼠标悬浮显示的内容
  • width 定义图片宽度
    • height 定义图片高度

超链接

a标签 即为超链接标签 可以定义一个超链接

a标签也可以作为锚点使用

a标签也可以用于发送邮件等其他功能

列表

列表是信息资源的一种展示方式 使信息结构条理更清晰 以便更快获得相应信息

列表主要分为:

  • 有序列表
  • 无序列表
  • 定义列表

有序列表(Order List) 用于有序排列项目

无序列表(Unorder List) 用于无序排列项目

定义列表(Definition List) 用于解释内容

表格

表格可以直观地展示数据

传统的HTML文档会使用table标签展示数据 后续会使用css来布局样式

table的单元格支持扩展

  • rowspan 行扩展 指此单元格占用几行 被占用的格子需要删除
  • colspan 列扩展 指此单元格占用几列 备战用的格子需要删除

媒体元素

用于展示媒体元素 src可以写本地也可以写网络地址 添加controls属性可以填加控制条 autoplay可以自动播放

表单

在前后端数据交互时 表单发挥了重要的作用 它可以将前端数据传递到后端

form 表单

  • method属性 定义form表单提交的方式 post / get
  • action属性 指定表单提交的目标位置/路径

input 输入框

  • id属性 指定input的id
  • name属性 指定input的键 后端获取需要对应
  • type属性 指定输入框类型
  • placeholder属性 在没有输入内容时 提醒用户输入

表单还可以添加基本的验证功能

当没有在验证范围内的时候 就无法提交表格

页面结构

HTML5 支持用语义展示页面结构 如下是HTML5支持的语义标签

元素名称 描述
header 标记头部内容区域
footer 标记脚部内容区域
section Web页面中独立的一个部分
article 独立的文章内容
aside 相关内容或者应用 (侧边栏)
nav 导航类辅助功能
