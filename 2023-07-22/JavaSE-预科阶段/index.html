博客

什么是博客

博客(Blog或WebLog) 即网络日记 在网络上出版, 发表和张贴的个人文章 通常以网页形式出现

在网络上发表Blog的构想始于1998年, 2000年才开始真正流行

目前有众多博客平台: 博客园 知乎 CSDN 简书等

博客的特点

  • 频繁更新Frequency
  • 简短明了Brevity
  • 个性化Personality

为何要写博客

博客主要具有如下用途

  • 个人自由表达和出版
  • 知识的过滤和积累 博客永远是共享与分享精神的体现
  • 深度交流的网络新方式
  • 博客营销

此外 对于个人而言 博客有如下用途

  • 作为网络个人日记承载形态
  • 个人展示的空间
  • 网络交友的平台
  • 抒发感情的树洞

本博客主要用于知识的总结和提炼并共享给各位看者

Markdown

什么是Markdown

Markdown 是一种轻量级标记语言 允许人们使用易读易写的纯文本格式编写文档

Markdown于2004年由John Gruber创建

Markdown文档后缀为.md或.markdown 且可以导出为HTML/Word/txt等诸多格式的文档

Github/Gitee仓库的readme.md文件通常为使用markdown语法的介绍该仓库的文件

Markdown特点

简单 美观 干净利落 所见即所得(可实时渲染)

Markdown编辑器

Markdown文档为纯文本文档 可以使用notepad VSCode Typora等诸多软件打开和编辑

在某些编辑器下 Markdown文档可以被渲染成美观的文档

笔者使用Typora作为Markdown编辑器 (0.x.x版本属于公测版本 不收费)

可以根据喜好和习惯使用不同的Markdown编辑器

Markdown基本语法

可以参考如下教程学习Markdown基础语法

  • 菜鸟教程 - Markdown基础教程
  • 简书 - Markdown基础语法

语法相对简单 请勤于尝试!

计算机

什么是计算机

计算机(computer)俗称电脑, 是现代一种用于高速计算的电子计算机器, 可以进行数值计算, 又可以进行逻辑计算, 还具有存储记忆功能. 是能够按照程序运行, 自动高速处理海量数据的现代化智能电子设备. 是20世纪最先进的科学技术发明之一

计算机由硬件和软件构成

计算机历史

  • 第1代 电子管数字机(1946~1958年) ENIAC
  • 第2代 晶体管数字机(1958~1964年) TRADIC
  • 第3代 集成电路数字机(1964~1970年)
  • 第4代 大规模集成电路计算机(1970年至今) Apple II

硬件层 冯诺依曼结构

冯·诺依曼结构也称普林斯顿结构 是一种将程序指令存储器和数据存储器合并在一起的存储器结构

当今世界的计算机绝大多数采用冯诺依曼结构

特点

  • 采用二进制逻辑
  • 程序存储执行 程序和数据存储不区分存储在存储器中
  • 指令顺序执行
  • 计算机由五个部分组成 以运算器为核心
  • 软硬件完全分离
  • …

软件层 软件分类

计算机软件主要分为系统软件和应用软件

Java即后续学习主要是面向企业/个人开发应用程序
