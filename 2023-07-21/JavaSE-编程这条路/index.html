编程语言

编程语言介绍

编程语言是人与计算机之间传递信息的媒介 能使用计算机帮助用户完成各种工作

计算机语言又分为Java, C, Python等诸多语言 (就好比现实有中文 英文 德语等诸多语言)

编程语言发展史

  1. 第一代语言 机器语言 (直接在纸带上打孔 交由计算机处理)
  2. 第二代语言 汇编语言 (利用汇编语言与计算机交互)
  3. 第三代语言 高级语言 C/C++ / Python / Java … (可以编译成文件交由计算机执行)

Java分为三个版本
​ - JavaSE Java标准版 针对桌面
​ - JavaME Java移动版 针对移动端
​ - JavaEE Java企业版 针对服务端

三高: 高性能 高并发 高可用

Java

什么是Java

Java是一种编程语言 被特意设计用于互联网的分布式环境

Java是由Sun公司在1995年推出的 目前最常用的是Java8/Java1.8 最新版(目前)为Java19

Java的历史

在1972年 C语言诞生

  • 贴近硬件 运行快 效率高
  • 操作系统 编译器 数据库 网络系统等
  • 指针和内存管理

在1982年 C++ 诞生

  • 面向对象
  • 兼容C
  • 图形领域 游戏等

很多复杂特性让学者望而却步 寻找一个新的编程语言 需要以下特性

  • 语法像C/C++
  • 没有指针和内存管理
  • 具有高可移植性
  • 面向对象 类型安全 具有高质量类库
  • …

至此 Java横空出世

Java的优势

  • 简单性: 语法简单 易于学习
  • 面向对象
  • 可移植性 一次编写到处运行(write once, run everywhere) 依赖于JVM(Java虚拟机)
  • 高性能 性能将会媲美C语言
  • 分布式
  • 动态性 反射
  • 多线程 更好的交互性
  • 安全性
  • 健壮性 异常机制
  • …

Java包含的内容

通常我们会从互联网上下载JDK (JDK包含了JRE JRE包含了JVM)

  • JDK: Java Development Kit, Java开发者工具
  • JRE: Java Runtime Environment, Java运行环境
  • JVM: Java Virturl Machine, Java虚拟机

JVM是Java代码可以在众多平台执行的根本原因 避免了物理平台的差异化

Java的下载安装在后续环境安装的文档中会具体说明

Java能做什么

Java几乎运行在生活的方方面面

  • 建立网站
  • 应用开发 (Android开发 企业级开发 桌面级开发等) Ant, Maven, Tomcat, Spring, Eclipse …
  • 嵌入式即消费类设备开发
  • 互联网大数据处理
  • 科学研究
  • 金融领域
  • … …

Java的分类

Java可以运行在众多平台上

  • JavaSE Java平台标准版 最多使用
  • JavaEE Java平台企业版 用于开发企业应用程序
  • JavaME Java平台微型版 用于开发嵌入式设备

如何学习Java

多写代码 多写笔记 多写文章

多练交流 多练思维 多练技能

多享知识 多问问题 多想为何

最后最重要的: 坚持 坚持 坚持!!!

Java的就业前景

Java仍然是当今世界主流编程语言之一 使用范围广 就业范围广 用于绝大多数领域的诸多方向 薪资较高

无论如何 尽量往高和深的方向学习 世界不缺CURD程序员 越高级越是难以被替代
