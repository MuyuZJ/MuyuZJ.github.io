什么是DB

DB 即Database 数据库 是长期存放并管理在计算机内有组织可共享的大量数据的”仓库”

数据库分为关系型数据库和非关系型数据库

关系型数据库和非关系型数据库

关系型数据库和非关系型数据库是数据库中两种不同的数据存储和管理方式

  • 关系型数据库
    • 典型的关系型数据库: MySQL / Oracle / Microsoft SQL Server
    • 关系型数据库是基于表格的形式存储数据 表格由行和列组成 每个单元格包含一个数据项
  • 非关系型数据库
    • 典型的非关系型数据库: Redis / MongoDB
    • 非关系型数据库则不使用表格的形式存储数据 而是使用键值对(key-value pairs)的形式存储数据

记录和字段

  • 记录是一个数据集中的一个具体的数据项 通常包含一个或多个字段
  • 字段是记录中用于存储数据的组成部分 它们具有特定的数据类型和属性

例如: 在一个学生数据表中

  • 一个学生有编号,姓名,年龄等诸多数据 每一个项目就是一个字段
  • 拥有这些数据项(字段) 的学生有很多个 则有多条记录

为何学习数据库

  • 数据库是存储数据的方式
  • 岗位技能需求
  • 大数据时代 的数据者得天下
  • 数据库是软件开发的核心之一
  • … …

什么是DBMS

DBMS 即DataBase Management System (数据库管理系统) 用于科学组织和存储数据 高效地获取和维护数据

本基础内容学习的就是MySQL数据库管理系统

什么是MySQL

MySQL是一个流行的开源免费的关系型数据库管理系统

由瑞典MySQL AB公司开发 目前属于Oracle旗下产品 MySQL官网

MySQL特点

  • 开源 免费
  • 体积小巧 基本功能齐全
  • 使用便捷 在线教程多
  • 可以用于Win/Mac/Linux 等多系统
  • 适用于中小型应用 甚至可以应用于部分大型网站

安装MySQL

MySQL可以从 MySQL官网 - 下载 - Community Server 中选择自己对应的系统版本进行下载

安装结束后可以使用命令行 输入命令mysql -u root -p 回车 以root用户的身份登录mysql 输入密码(默认不显示)

If you want to supply your password on the command line after the -p option, you must do so with no intervening space (for example, as -p*password*, not as -p *password*). However, putting your password on the command line is not recommended, because doing so exposes it to snooping by other users logged in on your machine.

结构化语句分类

名称 全称 解释 命令举例
DDL 数据库定义语言 定义和管理数据库对象 create drop alter
DML 数据库操作语言 操作数据库对象中的数据 insert update delete
DQL 数据库查询语言 查询数据库语句 select
DCL 数据库控制语言 管理数据库 管理权限和数据更改 grant commit rollback

使用上述语句可以对数据库进行操作

注意点:

  • Unix下 会区分大小写(Windows下不使用) 但建议使用小写字符作为数据库/表格/字段的名称
  • 当创建提示对数据库的访问被拒绝则需要考虑自己的权限是否足够

使用create关键字创建数据库 可以携带参数 (--开头的表示行注释)

  1. 数据库名称（database name）：这是必填参数，也是数据库的唯一标识符。
  2. 数据库字符集（character set）：指定数据库的字符集，如utf8、gbk等。
  3. 数据库校对（collation）：指定数据库的校对规则，如utf8_general_ci、gbk_chinese_ci等。
  4. 数据库权限（privileges）：指定数据库的权限，如SELECT、INSERT、UPDATE等。
  5. 数据库默认字符集（default charset）：指定数据库的默认字符集。
  6. 数据库默认校对（default collation）：指定数据库的默认校对规则。

文档:

How about age? That might be of interest, but it is not a good thing to store in a database. Age changes as time passes, which means you’d have to update your records often.

那么年龄(的存储)呢? 也许这很新奇 但是存储它不是一个好的方式. 年龄会随着时间的更改而变化 这意味着你需要经常更新这些记录

文档内容说明 实际情况下 存储年龄通常会存储出生时间或者时间字符串 以避免随时间的变化而带来的记录中的数据变化

drop命令请不要在不熟悉的数据库中随意使用!

IF EXISTS is used to prevent an error from occurring if the database or table does not exist.

IF EXISTS 被用于阻止当数据库或者数据表不存在的时候引发的错误

文档:

When a database is dropped, privileges granted specifically for the database are not automatically dropped. They must be dropped manually.
