注释

注释用于解释代码 是写给用户或者开发者 用于理解代码含义 计算机不执行的代码

写代码过程中书写关键代码的注释是一个非常好的习惯

注释分为单行注释 多行注释 和 文档注释

因为代码注释不会被计算机执行 故也产生了一些有趣的代码注释 CSDN - 有趣的代码注释

标识符具有一定的命名规范 Java命名规范 且不可与Java关键字同名

数据类型

Java是一种强类型的语言 每一个变量都具有各自对应的数据类型 (Python是弱类型语言)

如下是Java数据类型的分类

引用数据类型可以包含基本数据类型

数据类型之间可以转换 可能会丢失精度

在Java中不同精度数据可以混合运算 但是计算时候也需要线转换为同一数据类型再进行计算

数据类型转换注意点:

  • 不能对布尔类型的值进行转换
  • 不能转换不相干的类型
  • 低精度转高精度会自动转换 反之需要手动强制转换
  • 转换过程可能存在内存溢出或精度问题(无法准确表示)

变量

变量 即可以变化的量 是Java程序中最基本的存储单元 包括变量名,变量值和作用域

声明时必须指定数据类型 可以指定作用域 (分为 类变量 实例变量和局部变量)

  • 类变量 从属于类 为静态变量 在类初始化时即创建 默认为0
  • 实例变量 从属于对象 可以创建类对象后使用对象.变量名获得 默认为0
  • 局部变量 只针对局部区域 外部不可访问 默认为空

注意点:

  • 变量的类型可以是基本数据类型也可以是引用数据类型
  • 变量命名遵循Java标识符命名规则
  • 变量声明是一条语句 必须以分号分割
  • 只有当前能够访问到的变量才可以使用 否则报错 注意变量作用范围
  • 基本数据类型的变量无手动初始化时,执行默认初始化 默认为0或者0.0或者false … 引用数据类型为null

允许在一行定义并赋值多个变量

常量

常量 即不变化的量 初始化后则不能再改变 使用final关键字修饰

所谓常量可以理解为固定不变的变量 常量名通常使用大写表示 使用final关键字修饰

运算符

Java允许使用如下运算符

  • 算术运算符
    • + - * /
  • 赋值运算符
    • =
  • 关系运算符
    • > < >= <= == != instanceof
  • 逻辑运算符
    • && || !
  • 位运算符 (参考 菜鸟教程 - 位运算 )
    • & | ^ ~ >> << >>>
  • 条件运算符
    • ? :
  • 扩展赋值运算符
    • += -= *= /=
  • 自增自减运算符
    • ++ –

JavaDoc

在代码编写过程中 我们解释方法和类时会用到文档注释 是为了说明方法和生成JavaDoc

JavaDoc是可以将注释信息生成文档的工具 文档注释加在类上就是类文档注释 加载方法上就是方法文档注释

文档注释参数信息

  • @author 作者名
  • @version 版本号
  • @since 指定最早使用jdk的版本
  • @param 参数信息
  • @return 返回值情况
  • @throws 抛出异常情况
